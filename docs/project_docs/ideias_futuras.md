# Ideias Futuras para o Dashboard Clash Royale

## 1. Sistema de Maiores Rivais (Rivalry Dashboard)
Transformar a aba de "Oponentes Repetidos" em um painel estratégico de rivalidades.
- **Destaque Visual**: Cards para os Top 10 jogadores enfrentados mais vezes.
- **Head-to-Head (H2H)**: Placar direto detalhado (Vitórias vs Derrotas).
- **Categorização Automática**:
    - **Nêmese**: Oponentes com taxa de vitória > 60% contra o jogador.
    - **Freguês**: Oponentes que o jogador vence na maioria das vezes.
    - **Equilibrado**: Disputas pau a pau.
- **Timeline de Confrontos**: Pequeno gráfico ou lista de datas dos últimos encontros.

## 2. Visualização de Evoluções e Heróis
Melhorar a exibição de cartas especiais.
- **Evoluções**: Adicionar ícone ou borda especial para cartas evoluídas.
- **Heróis**: Garantir que as imagens dos heróis (Rainha Arqueira, Cavaleiro Dourado, etc.) estejam sempre atualizadas via CDN.

## 3. Análise de Sequência (Win/Loss Streaks)
Um gráfico que mostre as sequências de vitórias e derrotas ao longo do dia para identificar horários de "tilt".

## 4. Integração com DeckShop/RoyaleAPI
Links diretos para abrir os decks encontrados no dashboard diretamente no jogo ou em sites de estratégia.
